# Guide d'édition — Archive Marie-Renée Chevallier-Kervern

Pour l'équipe éditoriale. Aucune connaissance technique nécessaire — tout se fait
dans l'administration WordPress (`/wp-admin`).

## Se connecter

Rendez-vous sur `votre-site.fr/wp-admin` et connectez-vous. Votre compte est de
type **Éditeur** : vous gérez le contenu sans pouvoir modifier la configuration.

## Ajouter une œuvre

1. Menu **Archive → Ajouter**.
2. **Titre** de l'œuvre (ex. « Autoportrait »).
3. **Description** (facultatif) dans l'éditeur.
4. Bloc **« Fiche de l'œuvre »** : remplissez l'**année**, la **date affichée**
   (ex. « vers 1955 »), les **dimensions** (hauteur / largeur en cm + libellé
   affiché), le **support** (« huile sur toile »…), le **n° d'inventaire**, le
   **crédit**.
5. **Technique / Série / Thème / Collection** (colonne de droite) : cochez ou
   ajoutez un terme. Ces axes alimentent **automatiquement** les filtres du site.
6. **Image mise en avant** (colonne de droite) = l'image principale (vignette).
7. **Galerie d'images** : ajoutez les vues complémentaires (détails, verso…).
8. **Publier**.

> **Important (accessibilité) :** pour chaque image, renseignez le **texte
> alternatif** dans la médiathèque (décrivez brièvement l'œuvre). C'est une
> obligation légale pour un site public.

## Modifier la biographie (« La vie »)

Menu **Vie**. Chaque **chapitre** a un titre, un **sous-titre**, un texte (éditeur)
et une **galerie** d'images légendées. L'ordre des chapitres se règle via
**Attributs → Ordre**. Les légendes des photos = la **légende** de chaque image
dans la médiathèque.

## Modifier les pages (accueil, etc.)

Menu **Pages**. La page **Accueil** est la page d'accueil du site. Modifiez le
texte avec l'éditeur de blocs.

## Rechercher / vérifier

- Côté public : la page **Œuvres** offre recherche et filtres (technique, série,
  collection, année). Toute vue filtrée a sa propre adresse partageable.
- Côté admin : la liste **Archive** affiche vignette + année et se filtre par
  technique / série.

## Bonnes pratiques

- Utilisez un **n° d'inventaire unique** par œuvre (sert d'identifiant et évite
  les doublons lors des imports).
- Renseignez toujours le **texte alternatif** des images.
- Images : privilégiez de bonnes résolutions ; le site génère seul les formats
  d'affichage.
